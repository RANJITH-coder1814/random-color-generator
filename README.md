# random-color-generator
A simple web application that generates a random background color each time the user clicks a button. The app also displays the HEX color code and allows users to copy it to the clipboard.
# Random Color Generator

A simple web app that generates random colors and displays the HEX code.

## Features
- Generate random background color
- Display HEX color code
- Copy color code to clipboard

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Open index.html in your browser
